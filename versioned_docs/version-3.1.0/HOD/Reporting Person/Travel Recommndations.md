**Travel Recommendation and Approval by Reporting Person (RP)** 

- When an employee submits a travel request, the RP receives an email notification. 

- Log in to the HRMS portal. 

**For Recommendations**  

- From the sidebar, select **Travel**  

- Click on **Travel Recommendation List** 

**For Approvals** 

- From the sidebar, select Travel  

- Click on Travel Approval List 

- A list of travel requests from your team will appear with details such as: 

    - Travel ID (TR.ID) 

    - Employee Name 

    - Date of Request 

    - Project Name 

    - Trip Start Date & End Date 

    - Current Status (e.g., Verified by Admin, Recommended by TRP, Rejected by Admin) 

    - Locate the employee’s name in the list. 

Click on the forward arrow icon ( > ) next to the employee’s name to open the travel request details. 

After reviewing the request, you have options to: 

**Approve** - Approves the employee’s travel request. 

**Reject** - Rejects the employee’s travel request. 

Once the Reporting person takes action, the status column automatically updates, and the employee receives a notification.
