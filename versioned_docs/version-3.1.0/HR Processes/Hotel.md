The Hotel Module under the HR Profile is designed to manage employee accommodation requests during official travel.
It enables HR to raise, view, approve, or monitor hotel booking requests for themselves and other employees, ensuring smooth travel and stay arrangements.

**Hotel Request**

This section allows HR to submit their own hotel accommodation request during official travel.

**Steps to Create a Request**

- Go to the Hotel Request tab.

- Click Add New Request.

- Enter the required details such as:

- Travel Request ID (TR ID) – Select the related approved travel request.

- Location / City – Destination for accommodation.

- Check-in and Check-out Dates.

- Purpose of Stay.

Submit the request for approval.

**Hotel Booking History**

The Hotel History tab displays all hotel booking requests made by HR for their own travel.
It provides a quick overview of each request along with its check-in and check-out dates and the current approval status.

The Hotel Booking History page displays a list of all staff hotel booking requests submitted. 

Click the **>** icon to view complete booking details, including remarks and approval history.

**Staff Hotel Bookings**

The Staff Hotel Bookings page allows HR or administrative staff to view, track, and manage hotel booking requests made by employees during official travel or assignments.

**Functions and Features**

**Search by Name**: Use the search box at the top right corner to find a specific employee’s booking record quickly.

**Approved** – Booking has been confirmed.

**Cancelled** – Booking has been cancelled by signing authority or the employee.

**Pending** – Booking is awaiting approval (if applicable).

**Refresh Button (↻)**: Click to reload and update the booking list with the latest data.

**Navigation Button (← Hotel)**: Returns to the main Hotel dashboard or menu.

**Detailed View(›)**: Clicking the Details arrow icon opens a detailed booking summary.
