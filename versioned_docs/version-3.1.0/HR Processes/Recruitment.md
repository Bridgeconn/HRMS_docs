This tab is visible to HR users and helps manage recruitment related activities. 

- **Job Vacancy**
- **Refer A Friend** 

**Job Vacancy** - Expands to show sub-options for managing vacancies. 

- Add Vacancy
- Job Openings
- Job Applications

**Job Vacancy Submenu Options** 

**Add Vacancy** 

HR can create a new job vacancy with details such as title, department, experience, and closing date. 

**Job Opening Form – Field Descriptions** 

- Date – Automatically populates with current date.  

- Job Title (required) – Enter the name of the position. 

- Location (required) – Choose the job location from the dropdown list. 

- Salary – An optional field to mention the compensation offered for the role. 

- Role & Responsibilities – Provide a detailed explanation of the job duties and expectations. 

- Minimum Qualification (required) – Specify the least educational or professional qualification acceptable. 

- Required Skills – List the core competencies that are essential for the position. 

- Experience – Select the expected experience level from the available options. 

- Work Mode – Indicate whether the role is office-based or work-from-home. 

- Additional Skills – An optional section to add desirable or “nice-to-have” skills. 

- Last Date to Apply (required) – Choose the deadline for applications using the date picker. 

- Vacancy Status – Show the current state of the vacancy, such as Open, Closed, or On Hold. 

- Add Vacancy → Saves and publishes the job opening to the list of vacancies. 

**Job Openings**

The Job Openings page displays all recruitment vacancies posted by Bridge Connectivity Solutions.
This centralized view allows HR staff to manage and track all active and expired job postings.

**Key Features**

- Sortable columns for easy filtering
- Quick access to detailed job descriptions
- Direct editing capability for active management
- Clear visual indicators for expired positions using red text
- Clean tabular format for efficient scanning and review
- Details(▶) to view complete job description and requirements
- Edit(✎) to modify vacancy details, extend deadlines, or update information

**Job Applications**

The Job Applications page allows HR staff to review, manage, and process all applications received for posted vacancies. This is the central hub for candidate evaluation and selection workflow management.

The table displays all job applications with the following columns:
Applied Date, Self/Refer, Applied By / For, Referrer Id & Name, Job Title, Status

**Select/Reject**

- Green Checkmark (✓): Select/Approve the candidate
- Red Cross (✗): Reject the application

**Key Features**

- View and sort all applications in a centralized table
- Preview and download candidate CVs
- One-click approve or reject functionality with visual indicators
- Track referral sources and referrer details
- Real-time application status monitoring
- Distinguish between self-applied and referred candidates


**Refer A Friend** 

**Referral Form**: Submit new referrals

**Referral History**: Track previously submitted referrals

**Staff Referrals**: View referrals made by staff members
