The **Department** tab provides an overview of the company’s department-wise structure and heads. 

**Department-Wise Summary** - Displays the total number of employees in each department. 

**Department Heads’ Profile** - Shows department name, head of department, email ID, and profile photo (if uploaded). 

**Left Sidebar**

- Department Details - Manage main department data. 

- Department List - This screen displays all the departments created in the system along with their unique IDs. 

- Department ID - Unique identifier for each department (e.g., DPT01, DPT02). 

- Department - Name of the department (e.g., Software, Translation, Support Services). 

- Edit - Option to update department details. 

**Add a Department** 

- Click Add Department. 

- Enter required details (Department ID, Department Name). 

- Save to include it in the list. 

To **Edit** a Department,click the pencil icon in the Edit column. Modify department details as needed and save the changes.

**Sub Department Details**

This screen displays all the sub-departments mapped under their respective departments. 

**Available Actions** 

- Add a Sub Department 

- Click Add Sub Department. 

- Select the parent department. 

- Enter Sub Department ID and Name. 

- Save to include it in the list. 

**Edit a Sub Department** 

Click the pencil icon in the Edit column. Modify details as required and save the changes.

**Steps to modify** 

- Sub Department ID - Auto-generated unique code for the sub-department (e.g., SDT01). 

- Department - Dropdown to select the parent department (e.g., Management, Software, Support Services). 

- Sub Department - Enter the name of the sub-department (e.g., MGT, QA, Accounts). 

- Sub Department Head - Select the head responsible for the sub-department from the dropdown. 

- Sub Department Head Mail ID - Enter the email ID of the sub-department head. 

- Update Sub Department - Saves the updated details for an existing sub-department. 

- Cancel (X) - Closes the form without saving. 

 
**Designation Details**- Manage job roles/designations within departments. 

This page displays the list of all designations available within the organization. 

**Designation ID** → Unique ID assigned to each designation (e.g., DSG01, DSG02). 

**Department** - The department under which the designation belongs (e.g., Software, Management, Project Administration). 

**Designation** - The job title or role (e.g., Software Engineer, Managing Director, QA Executive). 

**Edit** - Pencil icon button to edit a specific designation. 

- Select Department (Dropdown) → Filter the list by selecting a specific department. 

- Refresh (⟳) → Reloads the designation list. 

- Add Designation (Top Menu) → Opens the form to create a new designation. 

**Edit Designation** 

- This page allows editing an existing designation’s details. 

- Designation ID → Auto-generated unique identifier (non-editable). Example: DSG01. 

- Department → Dropdown to select the department to which the designation belongs (e.g., Software, Management, Translation). 

- Designation → Text field for entering/updating the designation name (e.g., Software Engineer, Manager, QA Executive). 

Update Designation → Saves the updated details of the designation. 

1. **Add a New Designation** 

- Click on Add Designation. 

- Select the Department from the dropdown. 

- Enter the desired Designation title. 

- Click the Update Designation button (this will save the new designation). 

2. **Edit an Existing Designation** 

- Click on Designation List. 

- Select a designation to edit. 

- Modify the Department or Designation. 

- Click Update Designation to save the changes.
