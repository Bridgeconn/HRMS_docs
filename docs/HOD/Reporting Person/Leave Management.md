The **Reporting Person (RP)** is responsible for managing and monitoring the employees reporting to them. This role ensures smooth functioning of leave management, attendance approvals, timesheet tracking (for software department), providing inputs for employee appraisal and handling travel and hotel booking approvals. 

**Key Responsibilities of RP** 

**Leave Management** 

- Track leave balances of team members. 

- Review and approve/reject leave requests submitted by employees. 

**Steps to Approve/Reject Leave** 

- Employees initiate leave requests by applying through the HRMS platform.  

- Once submitted, the Reporting Person (RP) is notified via an email titled Leave Application for Approval.  

- The HRMS also displays an indication on the Notification icon, showing how many leave requests are available for review. 

**Login to HRMS** 

- Log in to the HRMS portal using your credentials. 

- Access Leave Applications 

- From the sidebar menu, click on **Leave.** 

- Select **Team Leave History**. 

- A page with all pending leave applications from your team will open. 

- Review the Application 

- Check the employeeâ€™s name, leave type, dates, and reason for leave. 

- To approve, click the **Approve** button. 

- To reject, click the **Reject button**. If the RP selects Reject, a pop-up window appears asking for the reason for rejection. The RP must enter the appropriate comment in the text box and then click Add to complete the action. 

- Bulk Approval Option (if multiple requests are pending) 
    - Select all leave applications by clicking the Select All button on top left-hand side  
    - Click on the Approve button to approve all selected requests at once. 

### Leave Report  
 
**Overview** 

The Leave Report feature allows authorized users to view and export employee leave details based on department, date range, and specific roles. The report can be generated on-screen and exported as an Excel file, which is delivered to the user via email. 
 
**Access Control** 

This feature is available only to users with specific roles/permissions configured in the HRMS system. 
 
**How to Access the Leave Report**

- Log in to the HRMS application using your credentials. 
- From the side navigation menu, click on Leave. 
- On the Leave page, click the Leave Report button. 

**Leave Report Filter** 
After clicking on the Leave Report, the Leave Report Filter screen opens. 

**Available Filters**

**Department** 
- Select the department for which the leave report is required. 
- Start Date 
- Choose the start date for the leave period. 
- End Date 
- Choose the end date for the leave period. 
- Employee Selection 
- Select individual employees from the list, or 
- Use Select All / Deselect All to quickly include or exclude all employees. 
- The employee list displayed is based on the selected department. 

**Generate Leave Report**

- After selecting the required filters, click Generate Report. 
- The system displays the Leave Report in a tabular format. 

**The report shows**

- Employee names 
- Dates 
- Leave types (e.g., CL, SL, EL, BL, SHL, Half-Day leaves) 
- Color-coded leave indicators for easy identification 
- Export Leave Report to Excel 
- On the Leave Report screen, click Export to Excel. 
- The system processes the request. 
- An email notification is sent to the logged-in user. 
- The email contains the Leave Report as an Excel attachment, including all selected filters and leave details. 

This ensures secure delivery and allows users to download and store the report for offline use.
